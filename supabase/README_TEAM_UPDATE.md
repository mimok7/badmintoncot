# 대기팀 기능 추가

## 데이터베이스 업데이트 필요

새로운 대기팀 기능을 사용하려면 Supabase 데이터베이스에 변경사항을 적용해야 합니다.

### 실행 방법

1. Supabase Dashboard에 로그인
2. SQL Editor로 이동
3. `supabase/add_team_number.sql` 파일의 내용을 복사하여 실행

### 변경 내용

- `reservations` 테이블에 `team_number` 컬럼 추가
- 각 대기팀은 최대 4명까지 제한
- 사용자는 같은 코트에서 하나의 팀에만 속할 수 있음
- 트리거 함수로 팀 인원 제한 자동 관리

### 새로운 기능

- 코트별로 여러 대기팀 생성 가능 (대기1, 대기2, 대기3...)
- 사용자가 원하는 대기팀 선택 가능
- 각 대기팀의 멤버 목록 실시간 표시
- 팀이 가득 차면 자동으로 다음 팀 생성 가능
